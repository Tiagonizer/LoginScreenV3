# LoginScreenV3

Tela de login para aplicativo Android desenvolvida com Kotlin e Android Studio.

## Sobre o projeto

Implementação de uma tela de login com validação de campos, 
desenvolvida como atividade da disciplina de Desenvolvimento Mobile.

## Tecnologias

- Kotlin
- Android Studio
- XML Layouts

## Contexto

Projeto acadêmico — Universidade de Fortaleza (UNIFOR)
